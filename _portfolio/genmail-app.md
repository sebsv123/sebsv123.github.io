## Genmail App

**genmail-app** es una herramienta web que genera emails profesionales y personalizados usando modelos de lenguaje de gran tamaño (LLMs).

### Características

- Generación de emails en múltiples tonos: formal, casual, persuasivo
- Plantillas para ventas, seguimiento, presentación profesional
- Personalización por nombre, empresa y contexto
- Interfaz minimalista y rápida

### Stack técnico

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** FastAPI (Python)
- **IA:** OpenAI API (GPT)
- **Despliegue:** Vercel (frontend), Railway (backend)
